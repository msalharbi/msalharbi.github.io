Introduction

This course introduces advance features in user interface design by studying both theoretical and practical issues in human-computer interfaces. These include development awareness of various approaches to the design of contemporary user interfaces, and to identify key principles by which effective contemporary user interfaces are designed. The course also addresses various interface issues that are applicable to interactive systems such as user-centered interaction thinking and methods of usability testing since there is a need to develop current and next-generation interactive technologies.

Grading

Midterms: 2 x 20 = 40

Practical Assessments: (two assignments x 7) + 6 = 20

  • Assignment 1 (7 points): Problem Definition, Affinity Diagram, and User Persona

    • This assignment involves defining a specific UI/UX problem, creating an affinity diagram to organize ideas and insights, and developing a user persona based on user research. The aim is to gain a deeper understanding of the users and their needs.

  • Assignment 2 (7 points): High-Fidelity Prototype using Figma

    • Students are required to design a high-fidelity prototype using Figma, incorporating design principles and user feedback. The prototype should demonstrate a well-structured user interface with interactive elements.

  • Assignment 3 (6 points): Data Gathering and Usability Evaluation

    • This final assignment involves collecting data through user feedback or other methods and conducting usability testing to evaluate the effectiveness of the prototype. Students will analyze the results and suggest improvements based on the gathered data.

Final Exam: 27 + 13 (Theoretical + Practical) = 40

Total: 100
