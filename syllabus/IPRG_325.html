Introduction

This course improves students' programming skills by emphasizing the importance of information hiding and abstraction in good program design. It covers basic data structures and algorithm analysis, teaching tools for problem-solving. Topics include algorithm analysis and design, estimation of algorithm growth rates using big O notation, and data structures such as lists, linked-lists, stacks, queues, priority queues, trees (with traversal), binary search trees, heaps, hash tables, and graphs, along with graph algorithms. Important sorting and searching algorithms are discussed, including bubble sort, insertion sort, selection sort, merge sort, heap sort, radix sort, sequential search, and binary search.
