  • إجرائية

أمثلة على لغات البرمجة

  • إجرائية: C، Pascal
  • كائنية: Java، C++
  • وظيفية: Haskell، Lisp

مقدمة عن البرمجة

البرمجة هي عملية إنشاء مجموعة من التعليمات التي تخبر الكمبيوتر بكيفية تنفيذ مهمة معينة.

أجزاء برنامج جافا بسيط


				public class Main {
					public static void main(String[] args) {
						System.out.println("Hello, World!");
					}
				}

أنواع أخطاء البرمجة

  • أخطاء بناء الجملة (Syntax Errors)
  • أخطاء وقت التشغيل (Runtime Errors)
  • الأخطاء المنطقية (Logical Errors)
  • الأخطاء الشائعة

أخطاء بناء الجملة (Syntax Errors)

هي الأخطاء في تركيب الكود التي تمنع البرنامج من الترجمة.

أخطاء وقت التشغيل (Runtime Errors)

هي الأخطاء التي تحدث أثناء تنفيذ البرنامج.

الأخطاء المنطقية (Logical Errors)

هي الأخطاء التي تحدث عندما يعمل البرنامج ولكنه ينتج نتائج غير صحيحة بسبب خلل في المنطق.

الأخطاء البرمجية الشائعة

  • نسيان الفواصل المنقوطة
  • أسماء المتغيرات غير صحيحة
  • استخدام خاطئ للمعاملات

الخاتمة

قدم هذا الفصل مقدمة إلى برمجة جافا، وتغطية أساسيات لغات البرمجة، وإعداد بيئة التطوير، والأخطاء البرمجية الشائعة.
